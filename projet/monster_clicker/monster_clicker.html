Monster Clicker :

Monster Clicker est un jeu réaliseé lors d'une jam en 2019 avec deux amis, le theme de cette jam était "mutation" et nous avons eu 7 jours pour réaliser ce jeu en python C'est un jeu qui mélange le concept d'un clicker comme le tres celebre "cookie clicker" et le concept de gatcha

Attention ce jeu n'a pas été balancé donc une fois un tirage effectué l'économie du jeu sera detruite, il n'a pas été optimisé non plus donc il se peut que le jeu crash, pour eviter les soucis penser a quitter le jeu pour sauvegarder
