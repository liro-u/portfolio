Chibi Project:

Chibi Project est un projet que je me suis lancé en aout 2021. C'est un jeu de combat en multijoueur.
Ce projet devait me faire découvrir le multijoueur et me permetre de m'ameliorer en 3d, je voulais également porter le jeu sur mobile.
(codé en 2021)

LANCER LE JEU:

Pour lancer le jeu vous devez lancer une version server sur un pc ou un téléphone.
Apres vous être connecté au meme wifi que le server vous devez récuperer l'adress ip du server avec par exemple la commande ipconfig dans le cmd.
Ensuite choisissez la map et le mode dans lequel vous voulez jouer depuis le server.
Puis une fois le server lancé, toutes les versions clientes entre l'ip du server et démarre le jeu.
Maintenant, amusez vous bien!

Le contenu :

Les cartes :

Pour le moment il n'y a que deux map, une qui m'a permit de faire des test sur le jeu pendant son developpement (la devmap), et une qui elle est plus pensé pour un vrai gameplay(fantasy world).

Les personnages:

A l'heure actuelle il n'y a que un personnage qui ne dispose que d'une attaque(touche j), et d'esquive(touche space), pour se deplacer vous pouvez faire avec Z,Q,S,D. Vous pouvez aussi utiliser les touches dédiés a la version mobile.

Les Objets:

Sur les maps vous pourrez rencontrer deux objets differents

Des barils qui explose quand vous tiré dessus idéal pour prendre l'aventage durant un combat.

Un healer qui vous redonne instantannément de la vie également pratique pour changer l'issus d'un combat mal engagé.
